# Shower Tray Research

## Target

Find a lightweight shower tray for a compact camper van shower cabin.

Available space: approximately 700 x 700 mm.

Preferred direction:

- lightweight ABS / thermoformed plastic for quick prototype validation
- custom composite / GRP tray for a more refined production solution if required
- suitable for installation with separate lightweight shower wall panels

## Shortlist

### Soltherm / Solera-Thermoform - TF00PD018

Product:

- Piatto doccia
- Code: TF00PD018

Known / observed data from supplier and reseller sources:

- approximate outer size reported as 72 x 68 cm by Soltherm
- reseller data often lists 665 x 665 x 80 mm
- usable inner shower area reported around 500 x 500 mm
- lightweight plastic / ABS shower tray for caravan and motorhome use
- drain prepared for vehicle-floor installation

Why it is interesting:

- good dimensional match for a 700 x 700 mm compact shower space
- very light camper-specific tray
- likely suitable for rapid CAD packaging and prototype evaluation

Open items:

- request STEP or other CAD data
- confirm exact geometry through model / technical drawing
- evaluate whether the usable standing area is acceptable for a premium camper concept
