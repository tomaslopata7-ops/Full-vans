# Email Template - Rotating Two-Seat Bench

Subject: Inquiry about rotating two-seat bench system for premium camper van project

Hello,

I am contacting you on behalf of KPL Design. Together with Full Vans, we are currently developing a new premium camper van interior.

We are looking for a safe and homologation-ready solution for a two-seat bench system that can be used in different interior positions. Please see the attached images for reference.

In the living / dining position, shown in the first attached image, the two-seat bench is positioned along the side wall of the vehicle and faces the table. In this position, the bench should sit as close to the side wall as possible in order to keep the interior space open and comfortable.

For driving, shown in the second attached image, the same seating system needs to rotate by 90 degrees so that the passengers sit facing forward, in the direction of travel. In this position, the seat must be securely locked and suitable for homologated passenger use in an M1 camper van.

The third attached image shows an alternative living / dining position. This position would be useful, but it is not strictly required.

Because the bench sits close to the side wall in the living position, we assume the mechanism would need to combine sliding and rotation:

- unlock the bench,
- slide it away from the side wall,
- rotate it by 90 degrees,
- lock it securely in the forward-facing driving position.

Ideally, the system should be lockable in the following positions:

- 0 degrees living / dining position, as shown in the first attached image,
- -90 degrees driving position, as shown in the second attached image,
- +90 degrees alternative living / dining position, as shown in the third attached image. This position would be useful, but it is not strictly required.

We are open to different solutions. This could be:

- a standalone swivel / sliding mechanism for a two-seat bench,
- a complete certified two-seat bench including the swivel / sliding base,
- or a custom-developed seating system including the bench, integrated seat belts, base frame and locking mechanism.

The most important requirements are:

- suitable for a two-seat bench,
- secure locking in the driving position,
- possible use in an M1 camper van,
- documentation for homologation / approval,
- preferably integrated 3-point seat belts,
- compact design suitable for a premium camper van interior.

1. Do you offer any product or system that could meet this requirement?
2. If not, would a custom or modified solution be possible from your side?
3. Can the system be locked in the 0 degrees living position and in the -90 degrees driving position? Is +90 degrees locking also possible as an optional alternative living position?
4. Are these locked positions covered by your ECE / TUV / homologation documentation, especially the -90 degrees driving position?
5. Could you also offer the complete seat / bench together with the rotating and sliding mechanism?
6. Would it be possible to receive technical documentation, dimensions, installation drawings and ideally 3D CAD data for packaging studies?

Our goal is to find a high-quality, safe and homologation-ready solution for a premium camper van project. We would be happy to discuss the technical requirements in more detail.
